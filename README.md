# 🏆 ContestPulse Widget

A sleek, premium desktop widget for tracking competitive programming contests in real-time. Built with Electron.

![Widget Showcase](https://raw.githubusercontent.com/Nareshkumawat-star/contestpulse/main/icon.png)

### ✨ Features
- **Modern 200x200 Square Tile**: A premium "pebble" design that looks great on any desktop.
- **Auto-Updates**: Users stay on the latest version automatically via GitHub Releases.
- **Rich Visuals**: Displays platform icons (Codeforces, CodeChef, etc.) and bold live countdowns.
- **Easy Drag & Move**: Dedicated drag zone at the top for effortless positioning without interfering with clicks.
- **Auto-Launch**: Automatically starts when you sign in to Windows.
- **Mobile Sync (Offline Support)**: Get notifications on your phone even when your PC is off via `ntfy.sh`.

### 🚀 Getting Started

#### For Development
1. Clone the repository.
2. Run `npm install`.
3. Run `npm start` to launch the widget.

#### For Distribution
- Run `npm run package` to create a standalone, shareable Windows application folder in `dist/`.
