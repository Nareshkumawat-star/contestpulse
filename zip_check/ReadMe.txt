🚀 CONTESTPULSE INSTALLATION GUIDE 🚀

To install the app on your computer:

1. Double-click the file named "Double_Click_To_Install.bat".
2. Wait for the green success message!

This will automatically:
- ✅ Create a Desktop Shortcut.
- ✅ Add it to your Start Menu.
- ✅ Add it to your "Installed Apps" list in Windows Settings.
- ✅ Set CodeChef, Codeforces, and LeetCode as defaults!

Enjoy your new coding contest widget! 🏆🏆🏆
